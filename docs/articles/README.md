`article_quantalayer_post_quantum_solana_research_note_1.pdf` is the canonical build filename. `research-note-1.pdf` is the stable public alias used by the title-page canonical URL. The two public PDFs must be byte-for-byte identical because both are copied from the same linearized build artifact. The official public checksum is the checksum of `research-note-1.pdf`.

The build artifact under `docs/articles/` may have a different SHA-256 hash from the public alias when the public alias is linearized for release. In that case, the official public hash is the hash in `public/articles/research-note-1.pdf.sha256`, while the build artifact hash is tracked separately in the release manifest and integrity audit.

Verification:

```bash
cd public/articles
sha256sum -c research-note-1.pdf.sha256
```
